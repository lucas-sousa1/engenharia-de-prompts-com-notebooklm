# 📖 Contexto e Objetivos

## Contexto

A Engenharia de Prompts tornou-se uma habilidade essencial para profissionais que utilizam Inteligência Artificial Generativa.

Este projeto foi desenvolvido durante o desafio da DIO para documentar os estudos realizados utilizando o NotebookLM como ferramenta de apoio ao aprendizado.

---

## Objetivos

- Aprender Engenharia de Prompts.
- Explorar diferentes técnicas de criação de prompts.
- Utilizar o NotebookLM para organizar informações.
- Documentar experimentos realizados.
- Construir um portfólio profissional no GitHub.

---

## Objetivo deste Projeto

O objetivo deste projeto é consolidar os conhecimentos adquiridos durante os estudos de Engenharia de Prompts, documentando técnicas, testes realizados, aprendizados e boas práticas de utilização da Inteligência Artificial.
