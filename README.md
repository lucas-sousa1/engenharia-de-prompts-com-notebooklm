# 📖 Sobre o Projeto

Este projeto foi desenvolvido durante o desafio da DIO com o objetivo de aprofundar os conhecimentos em Engenharia de Prompts utilizando o NotebookLM como ferramenta de apoio ao aprendizado.

Ao longo do projeto foram realizados experimentos, testes de prompts e organização do conhecimento adquirido.

---

# 🎯 Objetivos

- Estudar Engenharia de Prompts.
- Aprender técnicas de construção de prompts.
- Utilizar o NotebookLM para organização do conhecimento.
- Documentar experimentos realizados.
- Criar um projeto para portfólio no GitHub.
