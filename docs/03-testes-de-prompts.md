# 🧪 Testes de Prompts

## Objetivo

Documentar os principais experimentos realizados durante os estudos de Engenharia de Prompts utilizando o NotebookLM.

---

# Experimento 1

## Objetivo

Compreender o conceito de Engenharia de Prompt.

### Prompt Inicial

> Explique Engenharia de Prompt.

### Resultado

A resposta foi correta, porém muito genérica.

### Prompt Melhorado

> Explique Engenharia de Prompt para um estudante iniciante da área de Inteligência Artificial, utilizando exemplos práticos e linguagem simples.

### Resultado

A resposta ficou mais detalhada, didática e fácil de compreender.

### Aprendizado

Adicionar contexto e definir o público-alvo melhora significativamente a qualidade das respostas.

---

# Experimento 2

## Objetivo

Aplicar Engenharia de Prompt em empresas de tecnologia.

### Prompt Inicial

> Como utilizar Engenharia de Prompt em empresas?

### Resultado

A resposta apresentou aplicações gerais.

### Prompt Melhorado

> Como uma empresa de tecnologia pode utilizar Engenharia de Prompt para aumentar a produtividade das equipes de desenvolvimento, suporte e documentação?

### Resultado

Foram apresentadas aplicações práticas, como assistentes internos, geração de documentação técnica, automação de atendimento e apoio ao desenvolvimento de software.

### Aprendizado

Prompts específicos produzem respostas mais úteis e aplicáveis ao contexto empresarial.

---

# Conclusão

Durante os testes foi possível observar que pequenos ajustes nos prompts geram melhorias significativas na qualidade das respostas fornecidas pela Inteligência Artificial.
